# Agriculture Market Analyzer

Python tool for analyzing agricultural commodity market data across Indian states.

## Features
- Market-wise price distribution (animated)
- Price heatmaps
- Outlier detection
- Volatility analysis
- K-Means clustering
- Stability index

## Tech Stack
Python • Pandas • Scikit-learn • Matplotlib • Seaborn

## Usage
```bash
python agriculture_market_analyzer.py
```

## Dataset
10,000+ agricultural commodity records across 50+ commodities and 100+ markets
